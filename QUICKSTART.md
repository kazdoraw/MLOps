# Быстрый старт

## Структура готова!

Все файлы созданы и настроены. Теперь выполните шаги последовательно:

## Шаг 1: Запуск Jupyter Notebook

Откройте `hw1.ipynb` и **последовательно выполните ячейки**:

1. **Ячейка 2**: Установка зависимостей (может занять 2-3 минуты)
2. **Ячейка 5**: Загрузка датасета ChatDoctor с Kaggle
3. **Ячейка 7**: Анализ структуры данных
4. **Ячейка 8**: Инициализация Git и DVC
5. **Ячейка 10**: Добавление данных в DVC
6. **Ячейка 12**: Создание dvc.yaml
7. **Ячейка 14**: Запуск пайплайна `dvc repro` (обучение модели)
8. **Ячейка 16**: Фиксация в Git
9. **Ячейка 18**: Настройка DVC remote storage
10. **Ячейка 20**: Просмотр результатов MLflow

## Шаг 2: Просмотр результатов в MLflow UI

В терминале выполните:

```bash
cd /Users/Shared/ml/DEPLOY/HW1
mlflow ui --backend-store-uri sqlite:///mlflow.db
```

Откройте браузер: http://127.0.0.1:5000

## Шаг 3: Проверка воспроизводимости

```bash
dvc repro
```

Если изменений нет, увидите сообщение о том, что пайплайн актуален.

## Шаг 4: Эксперименты с параметрами

Измените параметры в `params.yaml`:

```yaml
train:
  model_type: "LogisticRegression"  # Попробуйте LogisticRegression
  n_estimators: 50                   # Или уменьшите для RF
  max_depth: 10                      # Измените глубину
```

Запустите снова:

```bash
dvc repro
```

MLflow автоматически залогирует новый эксперимент!
